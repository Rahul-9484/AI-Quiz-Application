##  Project Description

This is a web-based AI Quiz Application developed using Python Flask and SQLite. The system allows users to participate in quizzes and manage questions with role-based access.

##  Features

* User authentication (Login system)
* Role-based access (Admin, Teacher, Student)
* Add, edit, and delete quiz questions
* Attempt quizzes
* Automatic result calculation
* Performance tracking

##  Technologies Used

* Python
* Flask
* SQLite
* HTML, CSS, Bootstrap

##  Project Structure

* app.py → Main application file
* templates/ → HTML pages
* static/ → CSS and JS files
* database → SQLite database

##  How to Run the Project

1. Install Python
2. Install Flask:
   pip install flask
3. Run the application:
   python app.py
